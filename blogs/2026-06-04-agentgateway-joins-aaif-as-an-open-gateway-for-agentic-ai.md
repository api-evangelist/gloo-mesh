Agentgateway, an open source gateway built for modern AI systems, has joined the Agentic AI Foundation (AAIF) as its newest hosted project under the Linux Foundation. The project provides a unified layer for managing MCP traffic, Agent-to-Agent communication, LLM inference, REST APIs, and gRPC services through a single operational interface. With 300+ active contributors across 60+ organizations, agentgateway aims to address governance, observability, routing, and security challenges in increasingly complex agentic systems.
